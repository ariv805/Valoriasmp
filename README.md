# 🌲 VALORIA SMP
### Official Minecraft Server Website

![Minecraft](https://img.shields.io/badge/Minecraft-SMP-green)
![Website](https://img.shields.io/badge/Website-Next.js-black)
![Status](https://img.shields.io/badge/Server-Online-brightgreen)

Website resmi untuk server Minecraft **Valoria SMP**.  
Website ini dibuat untuk memberikan informasi server, vote system, galeri player, dan fitur komunitas lainnya.

---

# 🌐 Server Information

### 📡 IP Server
```
play.valoriasmp.my.id
```

### 🎮 Versi Minecraft
```
1.20+
```

### 🗺 Mode
```
Survival Multiplayer (SMP)
```

# ✨ Fitur Server

### 🛡 Land Claim
Melindungi bangunan player dari griefing.

### 👥 Teams System
Bermain bersama teman dalam satu tim.

### 🏆 Rank & Kits
Sistem rank dengan berbagai keuntungan.

### 🗳 Vote Rewards
Vote server dan dapatkan hadiah menarik.

### 🛒 Economy & Shop
Sistem ekonomi dan jual beli item.

### ⚔ PvP & Survival
Gameplay survival dengan PvP seru.

# 🚀 Instalasi

Clone repository:

```bash
git clone https://github.com/username/valoria-smp-website.git
```

Masuk ke folder project:

```bash
cd valoria-smp-website
```

Install dependencies:

```bash
npm install
```

Jalankan website:

```bash
npm run dev
```
